# LAN-Network-Implementation-and-Security
LAN Network Implementation and Security

Overview

This project demonstrates the implementation of a Local Area Network (LAN) using a Star Topology. The project focuses on configuring network connectivity, remote access, firewall rules, and basic network security techniques.

Features

Static IP Configuration
Ping Connectivity Testing
Remote Desktop Setup
Windows Firewall Configuration
MAC Address Filtering
Basic LAN Security

Technologies

Windows Networking
TeamViewer
BeeBEEP
Windows Defender Firewall
